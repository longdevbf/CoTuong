# Web CoTuongOffline

A web-based Xiangqi (Chinese Chess) game built with Next.js.  
Players can play with our AI bot or the other, view the movement rules of each piece, and interact with an intuitive chessboard interface.

## How to install
Clone repository:  
cd yourfolder (which cotains our game)  
open terminal  
git clone https://github.com/longdevbf/CoTuong.git  

Install dependencies:  
npm install  

Start game:  
npm run dev  

Run game at:  
http://localhost:8081
